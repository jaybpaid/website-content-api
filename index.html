Website Content API

Turn any URL into LLM‑ready content. Built on Apify's Website Content Crawler with adaptive scraping, schema.org extraction, and text chunking.

Pricing

  • Free: 50 requests/month (demo)
  • Starter: $9/month – 5,000 requests
  • Pro: $29/month – 25,000 requests
  • Enterprise: Custom – unlimited + SLA

Quick Start

curl -X POST https://api.example.com/scrape \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com", "chunk_size": 500}'
